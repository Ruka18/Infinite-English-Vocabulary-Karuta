# 無限かるた復元用プロンプト

## 現在の問題
- 画面が真っ白になってゲームが動作しない
- JSX構文エラーが解消できない
- HTMLファイルとJSXファイルの構造が破損している可能性

## 復元手順

### ステップ1：バックアップから復元
```
# 最新の動作していたバージョンから以下のファイルを復元：
1. mugen-karuta.jsx（メインのReactコンポーネント）
2. index.html（ブラウザ版）
```

### ステップ2：復元後の状態確認
- ゲーム画面が正常に表示されること
- すべてのボタンが動作すること
- カードがクリックできること

### ステップ3：最新の機能を再実装
以下の機能を復元したベースに追加：

#### 1. コンボシステム
- 現在のコンボ数表示（中央に1つ）
- 最大コンボ数の追跡
- リザルト画面への最大コンボ表示

#### 2. 効果音システム
- 8連続まで音階上昇（C4→C5）
- 8連続以降はC5を維持
- 盤面リセット時の音階リセットなし

#### 3. UI改善
- JP日本語・EN英語の表示削除
- GB→ENへの変更
- コンボ表示の中央配置

#### 4. その他機能
- 単語削除モード
- 日本語非表示モード
- リザルト画面の改善

### ステップ4：テスト
- ブラウザで正常に表示されること
- すべての機能が動作すること
- エラーがないこと

## 重要な注意点
1. まずは基本的なゲーム動作を確認
2. 機能を1つずつ追加してテスト
3. 各ステップで動作確認しながら進める
4. 問題が発生したらその時点で修正

## コマンド例
```bash
# HTTPサーバー起動してテスト
cd "c:\Users\toyah\Downloads\mugen-karuta"
python -m http.server 8080

# ブラウザで確認
http://localhost:8080
```

このプロンプトを使って、まずは安定したバージョンに戻し、そこから慎重に機能を再実装してください。
